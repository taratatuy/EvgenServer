Lines prefixed with:	Are sent from:		To:
<						client				server
>						server				connecting client


Get User
//Need to find user in database.

	< GET /getUser/?login=<login>&password=<password>

On true 
	> { getUser: true }
On false 
	> { getUser: false }
	


Create User
//Need to create user in database.

	< GET /getUser/?login=<login>&password=<password>

On true 
	> { createUser: true }
On false 
	> { createUser: false }
